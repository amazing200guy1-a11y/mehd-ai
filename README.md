Mehd AI is an enterprise-grade algorithmic execution engine powered by a distributed 11-agent neural network ('The Den').

Every single execution is **Consensus-Verified™**.

2. **The Brain (Logic Layer)**

## Architecture
- `backend/`: FastAPI + SSE data stream + HardRiskKernel + Sovereign Intelligence Data Moat.
- `flutter/`: Fully animated VS-Code aesthetic terminal, Compliance dashboard, Global Heatmap, and B2B Licensing Portal.

## Proprietary Features
- **Cascading Activation**: Reduces API overhead by dynamically escalating validation only for high-confidence setups.
- **The Chairman Agent**: Summarizes 11 agents into a 0-100 master score and Executive Brief.
- **Sovereign Intelligence**: Captures global context snapshots ('Alpha Snapshots') on high-consensus wins for future Llama finetuning.
- **Self-Correction Layer (Post-Mortem)**: Automatically amends the `app_constitution.json` if a high-confidence trade fails.
- **Consensus as a Service (CaaS)**: Allows 3rd-party prop firms to validate their edge via the `POST /api/consensus-validate` endpoint.

> **Legal & Compliance**: All Mehd AI internal processes are shielded by the immutable HardRiskKernel. Refer to the UI's Certificate of Intelligence for audit status. All trades are Consensus-Verified™.
