# Mehd AI — Institutional Forex Trading Terminal

Mehd AI is a premium, AI-powered trading assistant designed for the strategic trader. It focuses on **Capital Preservation** through a multi-model consensus engine and a strictly enforced HardRiskKernel.

## 🚀 Key Features

- **11 Specialized AI Agents**: The Den (Don, Phantom, Oracle, Caesar, Sage, Guardian, Titan, Atlas, Forge, The Don, Sentinel) analyze market sentiment, technicals, and math in real-time.
- **Zen Mode Charting**: A minimalist charting environment that strips away noise, focusing only on AI-generated zones and pure price action.
- **HardRiskKernel**: An unbreakable mathematical layer that enforces 1% risk per trade and prevents "Tilt" through an emotional firewall.
- **Multi-Language Support**: Full localization for 10 languages, including RTL support for Arabic.
- **Audit Trail**: Every decision, vote, and execution is logged to a permanent, immutable audit log.

## 🛠 Tech Stack

- **Frontend**: Flutter (Web/Mobile/Desktop) - Pure Black Premium UI.
- **Backend**: FastAPI (Python) - High-concurrency async architecture.
- **Deployment**: Firebase Hosting (Frontend PWA) & Railway (Backend).

## 🧰 Getting Started

### Backend
1. `cd backend`
2. `pip install -r requirements.txt`
3. `uvicorn main:app --reload`

### Frontend
1. `cd mehd_ai_flutter`
2. `flutter pub get`
3. `flutter run -d chrome`
